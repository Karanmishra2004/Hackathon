Explanation of Grid Properties used in this example:

  • Item 1: Spans 2 columns using `grid-column: span 2`.
  • Item 2: Starts on the second row using `grid-row-start: 2`.
  • Item 3: Centered within its grid cell using `place-items: center`.
  • Item 4: Occupies full width and half the height (2 rows) using `grid-template-rows: repeat(2, 1fr)`.
  • Item 5: Demonstrates spacing with `grid-gap: 20px` between items.
